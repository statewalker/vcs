## Core plumbing
* @statewalker/vcs-core
* @webrun-vcs/config
* @webrun-vcs/history
* @webrun-vcs/objects
* @webrun-vcs/refs
* @webrun-vcs/index
* @webrun-vcs/worktree

* @webrun-vcs/merge

## Wire & transport
* @webrun-vcs/protocol (Git + Fossil “sync”)
* @statewalker/vcs-transport

## Porcelain / features
* @webrun-vcs/repo
* @webrun-vcs/lfs

## Backends / env-specific

* @webrun-vcs/storage-fs
* @webrun-vcs/storage-sqlite
* @webrun-vcs/storage-idb / -opfs / -fsa etc.
* @webrun-vcs/worktree-* (node fs, browser FSA, OPFS)

Fossil-specific bits are mostly:

Binary delta engine in @tsgit/diff or @tsgit/delta.

Alternative object/container formats in @tsgit/objects.

Fossil sync protocol implementation in @tsgit/protocol.
