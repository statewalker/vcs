WebRTC P2P Git Sync

Peer-to-peer repository synchronization without a server

How It Works

  1. Peer A creates a repository and adds some files
  2. Peer A clicks "Create Offer" to generate connection data
  3. Copy the offer signal and paste it into Peer B's input
  4. Peer B clicks "Accept Offer" to generate an answer
  5. Copy the answer signal and paste it into Peer A's input
  6. Peer A clicks "Accept Answer" to complete the connection
  7. Once connected, click "Sync" to transfer the repository!
